# Hackathon_Doha_2025
Text-based adventure game for Hackathon Doha 2025

## Setup
Mac
```bash
curl -fsSL https://bun.com/install | bash
# You may need to open a new temrinal window
bun --version
# Output: 1.x.y
```
Windows
```bash
powershell -c "irm bun.sh/install.ps1|iex"
# You may need to open a new temrinal window
bun --version
# Output: 1.x.y
```

## Running
```bash
bunx --bun bun install
bunx --bun bun run dev
```
Open the localhost link that bun provides in a browser.
If cloning, you may need to import some extra videos for characters.
